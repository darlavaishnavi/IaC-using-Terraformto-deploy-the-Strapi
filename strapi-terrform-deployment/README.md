# Strapi Headless CMS Deployment on AWS using Terraform

This project contains Infrastructure as Code (IaC) using **Terraform** to deploy a
production-ready **Strapi CMS** on **AWS ECS Fargate**, with:

- ECS Fargate compute
- Application Load Balancer
- RDS PostgreSQL
- S3 Bucket for Media Files
- IAM Roles for secure access
- ECR for Docker Images
- GitHub Actions CI/CD
